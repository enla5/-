# -快递管理系统-项目导入IDEA即可
快递（包含取件和存件的相关操作）
### 1.该系统包含redis服务，需要开启redis服务端，可产生随机验证码并将验证码存入数据结构服务器。
### 2.数据库采用Mysql，项目采用jpa创建数据库表,只需要用户填入必填的几个快递柜位置和状态，其余都可在前端页面生成！
### 3.因为快递员信息创建的时候数量为零，所以填写信息时为0即可，后期会自动根据存入数量来填写。
### 4.网页采用LayUI设计
### 5.读者可自行下载修改
